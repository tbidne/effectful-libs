Produces a formatted LogStr in terms of:

  • Static Concurrent.
  • Static LoggerNS.
  • Dynamic Time.

Example

-- [timestamp][thread_label][namespace][code_loc][level] msg
[2022-02-08 10:20:05][thread-label][one.two][filename:1:2][Warn] msg
